# LeiTong02.github.io

Personal academic homepage and project pages for Lei Tong.

## Live Pages

- Homepage: https://leitong02.github.io/
- Causal-Adapter project page: https://leitong02.github.io/causaladapter/

## Repository Structure

- `index.html`: homepage entry
- `stylesheet.css`: homepage style
- `images/`: homepage assets
- `causaladapter/`: standalone project page (HTML/CSS/JS/assets)

## Update Workflow

1. Edit content in `index.html` (homepage) or `causaladapter/index.html` (project page).
2. Commit and push to `main` branch.
3. Wait for GitHub Pages deployment (usually within 1-3 minutes).
4. Verify the live URL.

## Notes

- This repo is published with GitHub Pages as a user site/project subpath setup.
- Keep relative paths inside `causaladapter/` unchanged so the page works under `/causaladapter/`.
