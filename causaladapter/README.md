# Causal-Adapter Project Page

Official project page for:

**Causal-Adapter: Taming Text-to-Image Diffusion for Faithful Counterfactual Generation**

Live URL: https://leitong02.github.io/causaladapter/

## Local Structure

- `index.html`: page content and sections
- `static/css/style.css`: visual style
- `static/js/main.js`: interactive behaviors
- `static/images/`: figures and thumbnails
- `static/videos/`: demo videos
- `static/pdfs/`: supplementary PDF assets

## How to Update

1. Edit content in `index.html`.
2. Add or replace assets under `static/`.
3. Commit and push to `main`.
4. Wait for GitHub Pages to redeploy.
5. Hard refresh the page (`Cmd+Shift+R`) if browser cache is stale.

## Troubleshooting

- If changes are not visible:
	- Confirm commit is pushed to `main`.
	- Check Actions/Pages build status on GitHub.
	- Hard refresh the browser.

- If 404 appears for the project page:
	- Verify URL is exactly `https://leitong02.github.io/causaladapter/`.
	- Ensure `causaladapter/index.html` exists.
	- Ensure Pages source is set to `main` and root.

- If commits appear under the wrong account:
	- Set repository local identity with `git config user.email` and `git config user.name`.
	- For HTTPS remotes, clear old GitHub credentials in macOS Keychain and sign in again.
